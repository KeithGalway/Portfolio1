Active Directory Glossary

Active Directory (AD)
A directory service developed by Microsoft for Windows domain networks.
Domain Controller
A server running the Active Directory Domain Services (AD DS) role.
Domain Services
The core functions in Active Directory that manage users and computers and allow sysadmins to organize the data into logical hierarchies.
Domain
A collection of objects within a Microsoft Active Directory network.
Schema
The set of user configured rules that govern objects and attributes in AD DS.
Global Catalog
The container of all objects in AD DS. If you need to find the name of a user, that name is stored in the Global Catalog.
Query and Index Mechanism
This system allows users to find each other in AD.

Active Directory

Active Directory stores information about objects on the network and makes this information easy for administrators and users to find and use.
